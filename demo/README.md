# CampusFlow Demo

这是一个无需安装依赖的静态 Web Demo，用于演示 CampusFlow 智校空间调度助手的四角色工作流：

1. 自然语言找空间。
2. 社团活动申请预审。
3. 老师审批工作台。
4. 管理员运营看板与周报摘要。

页面顶部提供 `3 分钟演示` 和 `10 分钟演示` 两种答辩模式，可按步骤切换学生、社团负责人、老师、管理员视图。

## 新增演示能力

- 答辩演示模式：顶部步骤条支持上一步、下一步和演示视图。
- AI 工作流卡片：默认展示“理解需求、查询数据、规则过滤、人工确认”，技术 JSON 折叠到细节中。
- 实时反馈：点击预约、反馈、通过、退回、调场后，右侧指标会联动变化。
- 数据与安全边界：结果区展示数据来源，并强调不接入个人轨迹、AI 不自动批准高风险申请。
- 投屏/移动适配：演示视图会收起两侧栏，移动端会变为单列布局。

## 打开方式

直接用浏览器打开 `demo/index.html`。页面使用普通本地脚本，不依赖构建工具或网络资源。

也可以在项目根目录启动任意静态服务器。例如：

```bash
python -m http.server 8080
```

然后访问：

```text
http://localhost:8080/demo/
```

## 文件说明

- `index.html`：Demo 页面结构。
- `styles.css`：页面样式。
- `src/engine.js`：意图解析、空间推荐、风险预审和运营摘要逻辑。
- `app.js`：页面 Mock 数据、交互和渲染。
- `tests/engine.test.mjs`：核心业务逻辑测试。
