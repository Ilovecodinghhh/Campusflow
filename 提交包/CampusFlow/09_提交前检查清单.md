## Demo 检查

提交前建议在本机执行：

```bash
cd 提交包/CampusFlow_V1.3_负责人提交包/10_初步实现源码
PYTHONPATH=apps/api python -m unittest discover -s apps/api/tests -v
PYTHONPATH=apps/api python -m campusflow.server
```

浏览器打开：

```text
http://127.0.0.1:8765
```

重点确认：

| 页面或接口 | 期望 |
| --- | --- |
| 首页标题 | `CampusFlow V1.3 试点交付版` |
| 管理员视图 | 显示 V1.3 试点交付看板 |
| `/api/health` | 返回 `status = ok` |
| `/api/pilot/delivery?role=管理员` | 返回 `version = V1.3` |
| V1.3 隐私字段 | `contains_customer_data = false` |
| V1.3 交付状态 | `delivery_status = ready_to_submit` |
