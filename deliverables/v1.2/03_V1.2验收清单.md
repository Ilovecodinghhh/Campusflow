# CampusFlow V1.2 验收清单

## 功能验收

| 项目 | 通过标准 |
| --- | --- |
| 模拟数据模块 | `apps/api/campusflow/pilot_data.py` 存在，并只包含独立模拟数据 |
| 隐私检查 | `contains_customer_data = false`，`forbidden_fields_found = []` |
| 准备度接口 | `/api/pilot/readiness?role=管理员` 返回 V1.2 报告 |

| 文档包 | `deliverables/v1.2/` 四份文档齐全 |

## API 验收

```bash
curl "http://127.0.0.1:8765/api/pilot/readiness?role=%E7%AE%A1%E7%90%86%E5%91%98"
```

关键字段：

| 字段 | 期望 |
| --- | --- |
| `version` | `V1.2` |
| `privacy.data_mode` | `independent_simulation` |
| `privacy.contains_customer_data` | `false` |
| `readiness_score` | 不低于 80 |
| `readiness_report.decision` | `ready_for_controlled_pilot` |
